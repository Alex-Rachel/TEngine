# Regicide 完全复刻验收清单

## 1. 规则一致性

- [ ] 牌局初始化符合配置（人数、手牌、敌方序列、随机种子）。
- [ ] 合法出牌能被服务端接收并推进结算。
- [ ] 非法出牌（非当前回合、越界索引）被拒绝且状态不变。
- [ ] 敌方未击败时触发反击并执行防御逻辑。
- [ ] 所有敌方击败后进入胜利结算。
- [ ] 触发必败条件后进入失败结算。

## 2. 联机一致性

- [ ] 2 人联机全流程（建房、就绪、开局、结算）通过。
- [ ] 3 人联机全流程通过。
- [ ] 4 人联机全流程通过。
- [ ] 客户端 UI 可交互状态与权威回合同步。
- [ ] 主机与客户端关键回合状态哈希一致。

## 3. 断线与异常

- [ ] 对局中断线重连后收到房间快照与最新战斗快照。
- [ ] 掉线玩家重连后可继续正常操作。
- [ ] 客户端异常退出后服务端完成席位清理。
- [ ] 对局结束后会话资源释放，不残留无效连接映射。

## 4. 资源与性能

- [ ] 战斗资源按清单异步加载（无同步阻塞）。
- [ ] 退出战斗后调用资源作用域释放，句柄可回收。
- [ ] 长局压测完成并记录指标（消息量、峰值帧耗时、GC 增量）。
- [ ] 无新增严重 Console Error（已知历史告警除外）。

## 5. 发布门槛

- [ ] 客户端构建通过。
- [ ] Dedicated Server 构建通过。
- [ ] 联调手册已执行复核并可复现部署流程。
- [ ] 故障排查手册关键步骤可用（连接失败、同步异常、重连失败）。
